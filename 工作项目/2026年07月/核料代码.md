### 2.7 普通BOM模式 `synInputAndOutOnStepByBOM`

```java
if (!this.enabledCheckInputMaterial()) {
    return
}
var semiProductIds = BillOfMaterial.querySemiProductIdsByBom(context())
Group<InputOnStep>().forEach{ it.isDirty = true }
Group<OutputOnStep>().forEach{ it.isDirty = true }
var toAddInput = new List<InputOnStep>()
var toAddOutput = new List<OutputOnStep>()
Group<WorkOrder>().groupBy{ it.productId + it.bomVer }.forEach((productIdAbomver,orders) -> {
    var firstOrder = orders.firstOrNull()
    if (firstOrder == null) {
        return
    }
    var productId = firstOrder.productId
    var ktRows = KT<KTInputConstraint>().select(it.productId == productId,it.stepSequenceNr != "")
    //1.0对有特殊约束的KT，在对应步骤创建input
    ktRows.forEach{ ktRow ->
        var boms = BillOfMaterial.queryByProductIdAndRawMaterialId(context(),productId,ktRow.rawMaterialId).filter{ it.isEffective }
        if (boms.isEmpty()) {
            return
        }
        indexing(WorkOrderStep.indexByProductIdAndSeqNr,productId,ktRow.stepSequenceNr).forEach{ workOrderStep ->
            if (workOrderStep == null) {
                return
            }
            var order = workOrderStep.owner
            if (order == null || !this.canUseBomByInputProductId(order,semiProductIds,ktRow.rawMaterialId)) {
                return
            }
            boms.forEach{ bom ->
                var input = indexing(InputOnStep.indexByOwnerAndMaterialIdAndLineNr,workOrderStep,bom.rawMaterialId,bom.lineNr).firstOrNull()
                if (input == null) {
                    input = new InputOnStep(workOrderStep)
                    toAddInput.add(input)
                }
                input.isDirty = false
                input.updateAttributeByBom(Utils.dataValuesToMap(bom))
            }
        }
    }
    //2.0根据产品代码查找BOM，并对普通BOM在第一步创建input
    var bomsData = BillOfMaterial.queryByProductId(context(),productId).filter{ it.isEffective }
    var hasConstraintRawMaterialIds = ktRows.map{ it.rawMaterialId }
    var commonBoms = bomsData.filter{ !hasConstraintRawMaterialIds.contain(it.rawMaterialId) }
    orders.forEach{ order ->
        indexing(Piece.indexByOwner,order).forEach{ piece ->
            var workOrderStep = piece.firstWorkOrderStep
            if (workOrderStep == null) {
                return
            }
            commonBoms.forEach{ bom ->
                if (!this.canUseBomByInputProductId(order,semiProductIds,bom.rawMaterialId)) {
                    return
                }
                var input = indexing(InputOnStep.indexByOwnerAndMaterialIdAndLineNr,workOrderStep,bom.rawMaterialId,bom.lineNr).firstOrNull()
                if (input == null) {
                    input = toAddInput.find{ it.owner == workOrderStep && it.materialId == bom.rawMaterialId && it.lineNr == bom.lineNr }
                }
                if (input == null) {
                    input = new InputOnStep(workOrderStep)
                    toAddInput.add(input)
                }
                input.isDirty = false
                input.updateAttributeByBom(Utils.dataValuesToMap(bom))
            }
        }
    }
    //3.0对于所有订单的所有末端步骤，创建output
    orders.forEach{ order ->
        var orderProductId = order.productId
        indexing(Piece.indexByOwner,order).forEach{ piece ->
            var workOrderStep = piece.selectedWorkOrderSteps.sortedBy{ it.sequenceNr }.lastOrNull()
            if (workOrderStep == null) {
                return
            }
            var output = indexing(OutputOnStep.indexByOwnerAndProductId,workOrderStep,orderProductId).firstOrNull()
            if (output == null) {
                output = toAddOutput.find{ it.owner == workOrderStep && Utils.anyToString(it.productId) == Utils.anyToString(orderProductId) }
            }
            if (output == null) {
                output = new OutputOnStep(workOrderStep)
                toAddOutput.add(output)
            }
            output.isDirty = false
            if (bomsData.isNotEmpty()) {
                output.updateAttributeByBom(Utils.dataValuesToMap(bomsData.firstOrNull()))
            }
        }
    }
})
Group<InputOnStep>().removeIf{ it.isDirty }
Group<OutputOnStep>().removeIf{ it.isDirty }
Group<InputOnStep>().addAll(toAddInput)
Group<OutputOnStep>().addAll(toAddOutput)
```

### 2.10 库存转供应 `synInventoryToMaterialSupply`

```java
var inventories = InventoryIn.queryAllData(context())
var toAdd = new List<MaterialSupply>()
var dh = context().getDbHelper()
var semiProductIds = BillOfMaterial.querySemiProductIdsByBom(context())
inventories.groupBy{ it.productId }.forEach((productId,inventoryList) -> {
    var product = indexing(Product.indexByProductId,productId).firstOrNull()
    var detectDuration = product?.detectDuration ?? Duration.ZERO
    var canSelfProduction = product?.canSelfProduction ?? false
    var isBanChengPing = semiProductIds.contain(productId)
    inventoryList.forEach{ inventory ->
        if (isBanChengPing) {
            var semiQty = inventory.quantity
            if (semiQty > Constant.EPS) {
                var obj = new MaterialSupply()
                MaterialSupply.updateAttributeBySemiInventoryNode(obj,inventory)
                obj.detectDuration = detectDuration
                obj.canSelfProduction = canSelfProduction
                this.setSemiFinishedInventoryDate(obj,inventory)
                this.setSemiProductVirtualQuantity(obj,productId)
                MaterialSupply.updateSupplySortKey(obj)
                if (obj.supplyQuantity > Constant.EPS) {
                    toAdd.add(obj)
                }
            }
            return
        }
        var availableQty = inventory.quantity
        if (availableQty > Constant.EPS) {
            var obj = new MaterialSupply()
            MaterialSupply.updateAttributeByInventoryAvailableNode(obj,inventory)
            obj.detectDuration = detectDuration
            obj.canSelfProduction = canSelfProduction
            obj.availableReleaseTime = obj.availableTime
            obj.availableReleaseDate = obj.availableDate
            MaterialSupply.updateSupplySortKey(obj)
            if (obj.supplyQuantity > Constant.EPS) {
                toAdd.add(obj)
            }
        }
        var detectQty = inventory.detectQuantity
        if (detectQty > Constant.EPS) {
            var obj = new MaterialSupply()
            MaterialSupply.updateAttributeByInventoryDetectNode(obj,inventory)
            obj.detectDuration = detectDuration
            obj.canSelfProduction = canSelfProduction
            obj.availableReleaseTime = TimeCalc.timePlusDuration(obj.availableTime,detectDuration)
            obj.availableReleaseDate = obj.availableReleaseTime.toDate()
            MaterialSupply.updateSupplySortKey(obj)
            if (obj.supplyQuantity > Constant.EPS) {
                toAdd.add(obj)
            }
        }
    }
})
dh.insertBatch(toAdd)
```

### 2.19 冻结分配重占 `reOccupyFrozenMaterialFulfill`

```java
var dh = context().getDbHelper()
var needUpdateDemands = new List<MaterialDemand>()
var needUpdateSupplies = new List<MaterialSupply>()
var invalidFrozenFulfills = new List<MaterialFulfill>()
var qfulfill = QMaterialFulfill.qmaterialfulfill
var frozenFulfills = dh.dslSelectFrom(qfulfill).where(qfulfill.frozenFlag.eq(true)).findAll()
var supplies = MaterialSupply.queryAllData(context())
var supplyMap = supplies.groupBy{ it.uniqueId }
frozenFulfills.groupBy{ it.demandUniqueId }.forEach((demandUniqueId,fulfills) -> {
    var demand = MaterialDemand.findByUniqueId(context(),demandUniqueId)
    if (demand == null) {
        invalidFrozenFulfills.addAll(fulfills)
        return
    }
    var firstPriorityDemand = MaterialDemand.findFirstReplacePriorityByMaterialDemand(context(),demand)
    var noLocationIds = MaterialSupply.queryNoLocationIdsByProductId(context(),demand.productId,demand.pieceSizeQuantity,demand.routingId)
    var isGuojiBanchengpingDemand = MaterialSupply.isGuojiBanchengpingMaterialDemand(demand,context())
    var handBooks = new List<KTProcessingTradeHandBook>()
    var candidateMaterialIds = new List<String>()
    if (isGuojiBanchengpingDemand) {
        handBooks = MaterialSupply.queryProcessingTradeHandBooks(context(),demand.productId,demand.materialId)
        candidateMaterialIds = handBooks.map{ it.materialId ?? "" }.filter{ it != "" }
        var demandMaterialId = demand.materialId ?? ""
        if (!candidateMaterialIds.contain(demandMaterialId)) {
            candidateMaterialIds.add(demandMaterialId)
        }
    }
    var hasValidFulfill = false
    fulfills.forEach{ fulfill ->
        var supply = supplyMap.getOrDefault(fulfill.supplyUniqueId,new List<MaterialSupply>()).firstOrNull()
        if (supply == null) {
            invalidFrozenFulfills.add(fulfill)
            return
        }
        //24039：校验冻结库存供应是否命中禁用库位
        var locationId = supply.sapStockingPointId ?? ""
        var invalidLocation = MaterialSupply.isInventorySupply(supply) && locationId != "" && noLocationIds.contain(locationId)
        if (invalidLocation) {
            invalidFrozenFulfills.add(fulfill)
            return
        }
        //24040：国际订单半成品校验手册物料和库位
        if (isGuojiBanchengpingDemand && !handBooks.isEmpty() && MaterialSupply.isInventorySupply(supply)) {
            var handBookSupplies = supplies.filter{ candidateMaterialIds.contain(it.productId ?? "") }
            var legalHandBookSupplies = new List<MaterialSupply>()
            handBookSupplies.forEach{ handBookSupply ->
                var handBookLocationId = handBookSupply.sapStockingPointId ?? ""
                var handBookInvalidLocation = MaterialSupply.isInventorySupply(handBookSupply) && handBookLocationId != "" && noLocationIds.contain(handBookLocationId)
                if (!handBookInvalidLocation) {
                    legalHandBookSupplies.add(handBookSupply)
                }
            }
            MaterialSupply.filterProcessingTradeHandBookSuppliesInPlace(legalHandBookSupplies,handBooks)
            if (!legalHandBookSupplies.contain(supply)) {
                invalidFrozenFulfills.add(fulfill)
                return
            }
        }
        demand.fulfillQuantity += fulfill.quantity
        hasValidFulfill = true
        if (firstPriorityDemand != null) {
            var demandQty = demand.pieceStepQuantity * demand.inputFactor
            var ratio = demandQty > 0.0 ? fulfill.quantity / demandQty : 0.0
            firstPriorityDemand.leftRatio -= ratio
        }
        supply.leftQuantity -= fulfill.quantity
        supply.leftQuantity.roundHalfDown(7)
        if (!needUpdateSupplies.contain(supply)) {
            needUpdateSupplies.add(supply)
        }
        fulfill.availableDate = supply.availableReleaseDate
        fulfill.availableTime = supply.availableReleaseTime
    }
    if (hasValidFulfill) {
        if (!needUpdateDemands.contain(demand)) {
            needUpdateDemands.add(demand)
        }
        if (firstPriorityDemand != null && firstPriorityDemand != demand && !needUpdateDemands.contain(firstPriorityDemand)) {
            needUpdateDemands.add(firstPriorityDemand)
        }
    }
})
if (!invalidFrozenFulfills.isEmpty()) {
    dh.deleteBatch(invalidFrozenFulfills)
}
var validFrozenFulfills = frozenFulfills.filter{ !invalidFrozenFulfills.contain(it) }
if (!needUpdateDemands.isEmpty()) {
    dh.updateBatch(needUpdateDemands)
}
if (!needUpdateSupplies.isEmpty()) {
    dh.updateBatch(needUpdateSupplies)
}
if (!validFrozenFulfills.isEmpty()) {
    dh.updateBatch(validFrozenFulfills)
}
```

    var demandQuantity = demand.demandQuantity

    var routingOf108= context.KT<KTProductRouting>().filter{it.productFrom108==true}

    var sourcePiece = context.Group<Piece>().find { it.pieceId == demand.pieceId }

    }

    var exactBatchCount = exactBatchCountMap.containsKey(guojiProductId)
        ? exactBatchCountMap.get(guojiProductId) : 0
    var maxBatchCount = maxBatchCountMap.containsKey(guojiProductId)
        ? maxBatchCountMap.get(guojiProductId) : 0

    var hasExactBatchLimit = exactBatchCount > 0
    var hasMaxBatchLimit = maxBatchCount > 0
    var guojiBatchCountLimit = hasExactBatchLimit ? exactBatchCount : maxBatchCount

    var targetBatchTotalQty = remainQty.roundHalfUp(6)
    var hasBatchInventory = !batchInventory.isEmpty()
    var hasInventorySupply = sortedSupplies.any { MaterialSupply.isInventorySupply(it) }
    var hasProductionSupply = sortedSupplies.any { MaterialSupply.isProductionPlanSupply(it) }
    var allInventorySupply = sortedSupplies.isNotEmpty() && sortedSupplies.all {
        MaterialSupply.isInventorySupply(it)

    // 国际订单成品 + BOM半成品 + 有批次数要求 + 有生产来源 + 有波动数量，才启用波动范围
    var hasUndulationBatchRange = isGuojiChengpingUseBomBanchengping

    // 波动范围基准量：优先使用成品订单数量 productQuantity
    var productQtyForUndulation = Utils.anyToDouble(demand.productQuantity).roundHalfUp(6)

    var minAllowedBatchTotalQty = productQtyForUndulation
    var maxAllowedBatchTotalQty = productQtyForUndulation

        maxAllowedBatchTotalQty = (productQtyForUndulation + undulationQuantity).roundHalfUp(6)

var hasLockedBatchResult = false//是否保留第一次分批结果

if (isGuojiChengpingUseBomBanchengping &&guojiBatchCountLimit > 0 &&guojiWorkOrder != null &&guojiWorkOrder.pieces != null &&guojiWorkOrder.pieces.isNotEmpty()) {

        batchCountMatched =

if (hasLockedBatchResult) {
    sourcePiece =context.Group<Piece>().find {it.pieceId == demand.pieceId}

    if (sourcePiece == null ||sourcePiece.owner == null) {

    var lockedMaterialUniqueId =Utils.anyToString(sourcePiece.selectedSemiProductUniqueId)

    if (lockedMaterialUniqueId.isEmpty()) {
        return tuple
    }

    /*

    var lockedSupplies =new List<MaterialSupply>()

    sortedSupplies.forEach { supply ->
        var supplyMaterialUniqueId =Utils.anyToString(supply.uniqueId)

        if (!supplyMaterialUniqueId.isEmpty() &&supplyMaterialUniqueId ==lockedMaterialUniqueId) {
            lockedSupplies.add(supply)
        }
    }

    if (lockedSupplies.isEmpty()) {
        return tuple
    }

    var lockedRemainQty =Utils.anyToDouble(sourcePiece.quantity).roundHalfUp(6)

    if (lockedRemainQty <= Constant.EPS) {
        return tuple
    }

    while (lockedSupplies.isNotEmpty() &&lockedRemainQty > Constant.EPS) {
        var supply =lockedSupplies.firstOrNull()

        if (supply == null) break

        var availableQty =MaterialSupply.getGuojiBanchengpingAvailableQty(supply).roundHalfUp(6)

        if (availableQty <= Constant.EPS) {
            lockedSupplies.remove(supply)
            continue
        }

        var qty =minOf(lockedRemainQty,availableQty).roundHalfUp(6)

        if (qty <= Constant.EPS) {
            lockedSupplies.remove(supply)
            continue
        }

        lockedRemainQty =(lockedRemainQty -qty).roundHalfUp(6)

        if (lockedRemainQty < 0.0) {
            lockedRemainQty = 0.0
        }

        var fulfillRatio =demandQuantity > Constant.EPS? qty / demandQuantity: 0.0

        totalFulfillRatio +=fulfillRatio

        if (!tempFlag) {
            var fulfill =MaterialFulfill.createFulfillByDemandAndSupply(demand,supply,qty)

            fulfill.ppProjectId =context.projectCode()

            fulfill.ppSceneId =context.sceneCode()

            fulfill.frozenFlag =frozenFlag

            fulfill.isSemiProductInPlant =isSemiProductInPlant

            fulfill.unFulfillQuantity =maxOf(lockedRemainQty, 0.0).roundHalfUp(6)

            toAdd.add(fulfill)

            demand.fulfillQuantity +=qty

            MaterialSupply.deductGuojiBanchengpingAvailableQty(supply,qty)

            if (firstDemand.isFirstConsumeSet) {
                firstDemand.leftRatio -=fulfillRatio
            }
        }

        if (MaterialSupply.getGuojiBanchengpingAvailableQty(supply) <= Constant.EPS) {
            lockedSupplies.remove(supply)
        }
    }

    /*
     * 无论绑定物料是否足量，都不允许重新拆批或切换其他物料。
     */
    return (totalFulfillRatio,toAdd)
}



    // 国际订单成品 + BOM半成品 + 有批次数要求
    if (isGuojiChengpingUseBomBanchengping
        && guojiBatchCountLimit > 0
        && targetBatchTotalQty > Constant.EPS) {

        var batchIndex = 1
        var createdBatchCount = 0
        var remainBatchQty = targetBatchTotalQty.roundHalfUp(6)
        var targetSingleBatchQty = (demand.productQuantity/ guojiBatchCountLimit).roundHalfUp(6)
        var frontBatchCount = guojiBatchCountLimit - 1

        // 前N-1批：循环中只记录每批数量
        while (sortedSupplies.isNotEmpty()
            && remainBatchQty > Constant.EPS
            && batchIndex <= frontBatchCount) {

            sortedSupplies = MaterialSupply.sortedForGuojiBatchCountDemand(
                sortedSupplies, targetSingleBatchQty
            )

            var supply = sortedSupplies.firstOrNull()
            if (supply == null) break

            var availableQty = MaterialSupply.getGuojiBanchengpingAvailableQty(supply)
            if (availableQty <= Constant.EPS) {
                sortedSupplies.remove(supply)
                continue
            }

            var batchQty = minOf(remainBatchQty, availableQty).roundHalfUp(6)
            if (batchQty <= Constant.EPS) {
                sortedSupplies.remove(supply)
                continue
            }

            remainBatchQty = (remainBatchQty - batchQty).roundHalfUp(6)
            if (remainBatchQty < 0.0) remainBatchQty = 0.0

            var fulfillRatio = demandQuantity > 0.0 ? batchQty / demandQuantity : 0.0
            totalFulfillRatio += fulfillRatio

            if (!tempFlag) {
                var fulfill = MaterialFulfill.createFulfillByDemandAndSupply(demand, supply, batchQty)
                fulfill.ppProjectId = context.projectCode()
                fulfill.ppSceneId = context.sceneCode()
                fulfill.frozenFlag = frozenFlag
                fulfill.isSemiProductInPlant = isSemiProductInPlant
                fulfill.unFulfillQuantity = maxOf(remainBatchQty, 0.0).roundHalfUp(6)
                toAdd.add(fulfill)

                demand.fulfillQuantity += batchQty
                MaterialSupply.deductGuojiBanchengpingAvailableQty(supply, batchQty)
                if (firstDemand.isFirstConsumeSet) firstDemand.leftRatio -= fulfillRatio
            }

            splitPieceQtyList.add(batchQty)
            splitPieceMaterialUniqueIdList.add(supply.uniqueId)
            createdBatchCount += 1
            batchIndex += 1
            sortedSupplies.remove(supply)
        }

    // 最后一批
    // 只有前 N-1 批处理完成后，仍然还有剩余批量需要处理时，才进入最后一批逻辑。
    if (remainBatchQty > Constant.EPS) {

        if (hasUndulationBatchRange) {

            // 而是从 sortedSupplies 中逐个选择一个半成品批量，

            // 或者在 productQuantity ± undulationQuantity 范围内
            var frontBatchTotalQty = 0.0
            splitPieceQtyList.forEach {

            }

            var idealLastBatchQty = (productQtyForUndulation - frontBatchTotalQty).roundHalfUp(6)

                    sortedSupplies.remove(supply)
                    continue
                }

                var equalProductQty = (availableQty - idealLastBatchQty)==0.0

                var canUseThisSupply = equalProductQty

                    canUseThisSupply = inUndulationRange
                }

                        demand.fulfillQuantity += qty
                        MaterialSupply.deductGuojiBanchengpingAvailableQty(supply, qty)

                        if (firstDemand.isFirstConsumeSet) firstDemand.leftRatio -= fulfillRatio
                    }

                    splitPieceQtyList.add(qty)
                    splitPieceMaterialUniqueIdList.add(supply.uniqueId)
                    createdBatchCount += 1

                    if (MaterialSupply.getGuojiBanchengpingAvailableQty(supply) <= Constant.EPS) {
                        sortedSupplies.remove(supply)
                    }

                    foundLastBatch = true
                    break
                }

                // 当前半成品不能使总批量等于 productQuantity，也不能落入波动范围，换下一个
                sortedSupplies.remove(supply)
            }

            if (!foundLastBatch) return tuple

        } else {
            // 无波动数量时，保留原来的最后一批逻辑： 多个供应共同凑齐 remainBatchQty。
            var lastBatchQty = remainBatchQty.roundHalfUp(6)
            var currentBatchRemainQty = lastBatchQty

                if (qty <= Constant.EPS) {
                    sortedSupplies.remove(supply)
                    continue

                if (remainBatchQty < 0.0) remainBatchQty = 0.0

                    fulfill.ppSceneId = context.sceneCode()
                    fulfill.frozenFlag = frozenFlag
                    fulfill.isSemiProductInPlant = isSemiProductInPlant
                    fulfill.unFulfillQuantity = maxOf(remainBatchQty, 0.0).roundHalfUp(6)
                    toAdd.add(fulfill)

                    demand.fulfillQuantity += qty
                    MaterialSupply.deductGuojiBanchengpingAvailableQty(supply, qty)
                    if (firstDemand.isFirstConsumeSet) firstDemand.leftRatio -= fulfillRatio
                }
                splitPieceMaterialUniqueIdList.add(supply.uniqueId)
                if (MaterialSupply.getGuojiBanchengpingAvailableQty(supply) <= Constant.EPS) {
                    sortedSupplies.remove(supply)
                }

            if (currentBatchRemainQty > Constant.EPS) return tuple

            splitPieceQtyList.add(lastBatchQty)
            
            createdBatchCount += 1
        }
    }
        // 批次数校验

                splitPieceMaterialUniqueIdList[
                    pieceIndex
                ]

                    if (pieceIndex == 0) {
                        // 第一次核料时，第一批复用原Piece
                        sourcePiece.quantity = calcPieceQty
                        sourcePiece.selectedSemiProductUniqueId =selectedMaterialUniqueId
                    } else {
                        // 第一次核料时创建剩余批次

                        var newPiece = workOrder.createNewPiece(calcPieceQty, maxSystemNum)
                        newPiece.selectedSemiProductUniqueId =selectedMaterialUniqueId
                        toAddPiece.add(newPiece)
                    }
                    pieceIndex = pieceIndex + 1
                }

            if (remainBatchQty < 0.0) remainBatchQty = 0.0
            var fulfillRatio = demandQuantity > Constant.EPS ? batchQty / demandQuantity : 0.0

            if (!tempFlag) {
                var fulfill = MaterialFulfill.createFulfillByDemandAndSupply(demand, supply, batchQty)
                fulfill.ppProjectId = context.projectCode()
                fulfill.ppSceneId = context.sceneCode()
                fulfill.frozenFlag = frozenFlag
                fulfill.isSemiProductInPlant = isSemiProductInPlant

var allInventorySupply = sortedSupplies.isNotEmpty() && sortedSupplies.all {
    MaterialSupply.isInventorySupply(it)
}

var productQtyForUndulation = Utils.anyToDouble(firstDemand.productQuantity).roundHalfUp(6)
if (productQtyForUndulation <= Constant.EPS) {
    productQtyForUndulation = targetBatchTotalQty
}

var hasUndulationBatchRange = isGuojiChengpingUseBomBanchengping
    && guojiBatchCountLimit > 0
    && hasProductionSupply
    && undulationQuantity > Constant.EPS

var minAllowedBatchTotalQty = productQtyForUndulation
var maxAllowedBatchTotalQty = productQtyForUndulation
if (hasUndulationBatchRange) {
    minAllowedBatchTotalQty = maxOf(
        0.0,
        productQtyForUndulation - undulationQuantity
    ).roundHalfUp(6)
    maxAllowedBatchTotalQty = (
        productQtyForUndulation + undulationQuantity
    ).roundHalfUp(6)
}

/*
 * 第二次及后续核料：复用第一次拆批结果。
 */
var hasLockedBatchResult = false
if (isGuojiChengpingUseBomBanchengping
    && guojiBatchCountLimit > 0
    && guojiWorkOrder != null
    && guojiWorkOrder.pieces != null
    && guojiWorkOrder.pieces.isNotEmpty()) {

    var batchCountMatched = false

    if (hasExactBatchLimit) {
        batchCountMatched = guojiWorkOrder.pieces.size() == exactBatchCount
    } else if (hasMaxBatchLimit) {
        batchCountMatched = guojiWorkOrder.pieces.size() > 0
            && guojiWorkOrder.pieces.size() <= maxBatchCount
    }

    var allPiecesHaveLockedMaterial = true
    guojiWorkOrder.pieces.forEach { piece ->
        var isPieceLocked = piece.lockPiece ?? true
        var lockedMaterialUniqueId = Utils.anyToString(piece.selectedSemiProductUniqueId)
        if (isPieceLocked != true || lockedMaterialUniqueId.isEmpty()) {
            allPiecesHaveLockedMaterial = false
        }
    }

    hasLockedBatchResult = batchCountMatched && allPiecesHaveLockedMaterial
}

if (hasLockedBatchResult) {
    sourcePiece = context.Group<Piece>().find { it.pieceId == firstDemand.pieceId }
    if (sourcePiece == null || sourcePiece.owner == null) return tuple

    var lockedMaterialUniqueId = Utils.anyToString(sourcePiece.selectedSemiProductUniqueId)
    if (lockedMaterialUniqueId.isEmpty()) return tuple

    var lockedSupplies = new List<MaterialSupply>()
    sortedSupplies.forEach { supply ->
        var supplyMaterialUniqueId = Utils.anyToString(supply.uniqueId)
        if (!supplyMaterialUniqueId.isEmpty()
            && supplyMaterialUniqueId == lockedMaterialUniqueId) {
            lockedSupplies.add(supply)
        }
    }

    if (lockedSupplies.isEmpty()) return tuple

    var lockedRemainQty = Utils.anyToDouble(sourcePiece.quantity).roundHalfUp(6)
    if (lockedRemainQty <= Constant.EPS) return tuple

    while (lockedSupplies.isNotEmpty() && lockedRemainQty > Constant.EPS) {
        var supply = lockedSupplies.firstOrNull()
        if (supply == null) break

        var currentDemand = demandMap.containsKey(Utils.anyToString(supply.productId))
            ? demandMap.get(Utils.anyToString(supply.productId))
            : null
        if (currentDemand == null) return tuple

        var availableQty = MaterialSupply.getGuojiBanchengpingAvailableQty(supply).roundHalfUp(6)
        if (availableQty <= Constant.EPS) {
            lockedSupplies.remove(supply)
            continue
        }

        var qty = minOf(lockedRemainQty, availableQty).roundHalfUp(6)
        if (qty <= Constant.EPS) {
            lockedSupplies.remove(supply)
            continue
        }

        lockedRemainQty = (lockedRemainQty - qty).roundHalfUp(6)
        if (lockedRemainQty < 0.0) lockedRemainQty = 0.0

        var currentDemandQuantity = currentDemand.demandQuantity
        if (currentDemandQuantity <= Constant.EPS) currentDemandQuantity = demandQuantity
        var fulfillRatio = currentDemandQuantity > Constant.EPS
            ? qty / currentDemandQuantity
            : 0.0
        totalFulfillRatio += fulfillRatio

        if (!tempFlag) {
            var fulfill = MaterialFulfill.createFulfillByDemandAndSupply(
                currentDemand,
                supply,
                qty
            )
            fulfill.ppProjectId = context.projectCode()
            fulfill.ppSceneId = context.sceneCode()
            fulfill.frozenFlag = frozenFlag
            fulfill.isSemiProductInPlant = bomBanchengpingIds.contain(
                Utils.anyToString(currentDemand.materialId)
            )
            fulfill.unFulfillQuantity = maxOf(lockedRemainQty, 0.0).roundHalfUp(6)
            toAdd.add(fulfill)

            currentDemand.fulfillQuantity += qty
            MaterialSupply.deductGuojiBanchengpingAvailableQty(supply, qty)

            if (firstDemand.isFirstConsumeSet) {
                firstDemand.leftRatio -= fulfillRatio
                if (firstDemand.leftRatio < Constant.EPS) firstDemand.leftRatio = 0.0
            }
        }

        if (MaterialSupply.getGuojiBanchengpingAvailableQty(supply) <= Constant.EPS) {
            lockedSupplies.remove(supply)
        }
    }

    return (totalFulfillRatio, toAdd)
}

/*
 * 国际订单成品 + BOM半成品 + 有批次数要求。
 * 先预演完整拆批结果，校验成功后再统一创建 fulfill 和扣减 supply，
 * 避免批次数校验失败时已经修改库存。

if (guojiBatchCountLimit > 0 && targetBatchTotalQty > Constant.EPS) {
    var batchSupplyList = new List<MaterialSupply>()
    var batchDemandList = new List<MaterialDemand>()
    var batchQtyList = new List<Double>()

    var previewSupplies = new List<MaterialSupply>()
    previewSupplies.addAll(sortedSupplies)

    var previewRemainBatchQty = targetBatchTotalQty.roundHalfUp(6)
    var productQtyBase = Utils.anyToDouble(firstDemand.productQuantity)
    if (productQtyBase <= Constant.EPS) productQtyBase = targetBatchTotalQty
    var targetSingleBatchQty = (productQtyBase / guojiBatchCountLimit).roundHalfUp(6)
    var frontBatchCount = guojiBatchCountLimit - 1
    var batchIndex = 1

    /*
     * 前 N-1 批。
     */
    while (previewSupplies.isNotEmpty()
        && previewRemainBatchQty > Constant.EPS
        && batchIndex <= frontBatchCount) {

        // var batchSortedSupplies = new List<MaterialSupply>()
        // sortedDemands.forEach { currentDemand ->
        //     var currentMaterialSupplies = previewSupplies.filter { supply ->
        //         Utils.anyToString(supply.productId)
        //             == Utils.anyToString(currentDemand.materialId)
        //     }
        //     currentMaterialSupplies = MaterialSupply.sortedForGuojiBatchCountDemand(
        //         currentMaterialSupplies,
        //         targetSingleBatchQty
        //     )
        //     batchSortedSupplies.addAll(currentMaterialSupplies)
        // }
        // previewSupplies = batchSortedSupplies

        var supply = previewSupplies.firstOrNull()
        if (supply == null) break
        previewSupplies.remove(supply)

        var currentDemand = demandMap.containsKey(Utils.anyToString(supply.productId))
            ? demandMap.get(Utils.anyToString(supply.productId))
            : null
        if (currentDemand == null) continue

        var availableQty = MaterialSupply.getGuojiBanchengpingAvailableQty(supply).roundHalfUp(6)
        if (availableQty <= Constant.EPS) continue

        var batchQty = minOf(previewRemainBatchQty, availableQty).roundHalfUp(6)
        if (batchQty <= Constant.EPS) continue

        batchSupplyList.add(supply)
        batchDemandList.add(currentDemand)
        batchQtyList.add(batchQty)

        previewRemainBatchQty = (previewRemainBatchQty - batchQty).roundHalfUp(6)
        if (previewRemainBatchQty < 0.0) previewRemainBatchQty = 0.0
        batchIndex += 1
    }

    /*
     * 最后一批必须由一条 supply 对应，确保一个 Piece 只绑定一个 uniqueId。
     */
    if (previewRemainBatchQty > Constant.EPS) {
        var frontBatchTotalQty = 0.0
        batchQtyList.forEach { frontBatchTotalQty += it }
        frontBatchTotalQty = frontBatchTotalQty.roundHalfUp(6)

        var foundLastBatch = false

        while (previewSupplies.isNotEmpty()) {
            var supply = previewSupplies.firstOrNull()
            if (supply == null) break
            previewSupplies.remove(supply)

            var currentDemand = demandMap.containsKey(Utils.anyToString(supply.productId))
                ? demandMap.get(Utils.anyToString(supply.productId))
                : null
            if (currentDemand == null) continue

            var availableQty = MaterialSupply.getGuojiBanchengpingAvailableQty(supply).roundHalfUp(6)
            if (availableQty <= Constant.EPS) continue

            var lastBatchQty = 0.0
            var canUseThisSupply = false

            if (hasUndulationBatchRange) {
                var idealLastBatchQty = (
                    productQtyForUndulation - frontBatchTotalQty
                ).roundHalfUp(6)
                var candidateTotalQty = (
                    frontBatchTotalQty + availableQty
                ).roundHalfUp(6)

                var qtyDiff = availableQty - idealLastBatchQty
                var equalProductQty = qtyDiff <= Constant.EPS && qtyDiff >= -Constant.EPS
                var inUndulationRange =
                    candidateTotalQty + Constant.EPS >= minAllowedBatchTotalQty
                    && candidateTotalQty <= maxAllowedBatchTotalQty + Constant.EPS

                canUseThisSupply = equalProductQty || inUndulationRange
                if (canUseThisSupply) {
                    lastBatchQty = availableQty.roundHalfUp(6)
                }
            } else {
                canUseThisSupply = availableQty + Constant.EPS >= previewRemainBatchQty
                if (canUseThisSupply) {
                    lastBatchQty = previewRemainBatchQty.roundHalfUp(6)
                }
            }

            if (!canUseThisSupply || lastBatchQty <= Constant.EPS) continue

            batchSupplyList.add(supply)
            batchDemandList.add(currentDemand)
            batchQtyList.add(lastBatchQty)
            previewRemainBatchQty = 0.0
            foundLastBatch = true
            break
        }

        if (!foundLastBatch) return tuple
    }

    var createdBatchCount = batchQtyList.size()
    if (hasExactBatchLimit && createdBatchCount != exactBatchCount) return tuple
    if (hasMaxBatchLimit && createdBatchCount > maxBatchCount) return tuple
    if (previewRemainBatchQty > Constant.EPS) return tuple

    var commitRemainQty = targetBatchTotalQty.roundHalfUp(6)
    var commitIndex = 0

    batchSupplyList.forEach { supply ->
        var currentDemand = batchDemandList[commitIndex]
        var batchQty = batchQtyList[commitIndex]

        commitRemainQty = (commitRemainQty - batchQty).roundHalfUp(6)
        if (commitRemainQty < 0.0) commitRemainQty = 0.0

        var currentDemandQuantity = currentDemand.demandQuantity
        if (currentDemandQuantity <= Constant.EPS) currentDemandQuantity = demandQuantity
        var fulfillRatio = currentDemandQuantity > Constant.EPS
            ? batchQty / currentDemandQuantity
            : 0.0
        totalFulfillRatio += fulfillRatio

        if (!tempFlag) {
            var fulfill = MaterialFulfill.createFulfillByDemandAndSupply(
                currentDemand,
                supply,
                batchQty
            )
            fulfill.ppProjectId = context.projectCode()
            fulfill.ppSceneId = context.sceneCode()
            fulfill.frozenFlag = frozenFlag
            fulfill.isSemiProductInPlant = bomBanchengpingIds.contain(
                Utils.anyToString(currentDemand.materialId)
            )
            fulfill.unFulfillQuantity = maxOf(commitRemainQty, 0.0).roundHalfUp(6)
            toAdd.add(fulfill)

            currentDemand.fulfillQuantity += batchQty
            MaterialSupply.deductGuojiBanchengpingAvailableQty(supply, batchQty)

            if (firstDemand.isFirstConsumeSet) {
                firstDemand.leftRatio -= fulfillRatio
                if (firstDemand.leftRatio < Constant.EPS) firstDemand.leftRatio = 0.0
            }
        }

        splitPieceQtyList.add(batchQty)
        splitPieceMaterialUniqueIdList.add(Utils.anyToString(supply.uniqueId))
        commitIndex += 1
    }

    /*
     * 创建/更新 Piece，并保存每批实际使用的半成品 uniqueId。
     */
    if (!tempFlag && splitPieceQtyList.isNotEmpty()) {
        sourcePiece = context.Group<Piece>().find { it.pieceId == firstDemand.pieceId }
        if (sourcePiece == null || sourcePiece.owner == null) return tuple

        var workOrder = sourcePiece.owner
        var currentPieceCount = workOrder.pieces.size()

        if (currentPieceCount == guojiBatchCountLimit) {
            return (totalFulfillRatio, toAdd)
        }

        var maxSystemNum = workOrder.pieces.maxByOrNull {
            it.systemNum ?? 0
        }?.systemNum ?? 0

        var pieceIndex = 0
        splitPieceQtyList.forEach { pieceQty ->
            var calcPieceQty = pieceQty.roundHalfUp(6)
            var selectedMaterialUniqueId = splitPieceMaterialUniqueIdList[pieceIndex]

            if (calcPieceQty > Constant.EPS && !selectedMaterialUniqueId.isEmpty()) {
                if (pieceIndex == 0) {
                    sourcePiece.quantity = calcPieceQty
                    sourcePiece.selectedSemiProductUniqueId = selectedMaterialUniqueId
                } else {
                    maxSystemNum += 1
                    var newPiece = workOrder.createNewPiece(calcPieceQty, maxSystemNum)
                    newPiece.selectedSemiProductUniqueId = selectedMaterialUniqueId
                    toAddPiece.add(newPiece)
                }
            }
            pieceIndex += 1
        }

        if (toAddPiece.isNotEmpty()) {
            context.Group<Piece>().addAll(toAddPiece)
        }

        PlanManager.getInstance(context).splitPiecesToPieceSteps()
    }

    return (totalFulfillRatio, toAdd)
}

/*
 * 国际订单成品未指定批次数：
 * 按库存A -> 库存B -> 生产A -> 生产B逐批分配；
 * 库存分完仍不足时允许继续使用生产。
 */
if (guojiBatchCountLimit <= 0 && remainQty > Constant.EPS) {
    sourcePiece = context.Group<Piece>().find { it.pieceId == firstDemand.pieceId }
    if (sourcePiece == null || sourcePiece.owner == null) return tuple

    var workOrder = sourcePiece.owner
    var batchSupplyList = new List<MaterialSupply>()
    var batchDemandList = new List<MaterialDemand>()
    var batchQtyList = new List<Double>()

    var previewSupplies = new List<MaterialSupply>()
    previewSupplies.addAll(sortedSupplies)
    var previewRemainQty = remainQty.roundHalfUp(6)

    while (previewSupplies.isNotEmpty() && previewRemainQty > Constant.EPS) {
        var supply = previewSupplies.firstOrNull()
        if (supply == null) break
        previewSupplies.remove(supply)

        var currentDemand = demandMap.containsKey(Utils.anyToString(supply.productId))
            ? demandMap.get(Utils.anyToString(supply.productId))
            : null
        if (currentDemand == null) continue

        var availableQty = MaterialSupply.getGuojiBanchengpingAvailableQty(supply).roundHalfUp(6)
        if (availableQty <= Constant.EPS) continue

        var batchQty = minOf(previewRemainQty, availableQty).roundHalfUp(6)
        if (batchQty <= Constant.EPS) continue

        batchSupplyList.add(supply)
        batchDemandList.add(currentDemand)
        batchQtyList.add(batchQty)

        previewRemainQty = (previewRemainQty - batchQty).roundHalfUp(6)
        if (previewRemainQty < 0.0) previewRemainQty = 0.0
    }

    if (previewRemainQty > Constant.EPS) return tuple

    var remainBatchQty = remainQty.roundHalfUp(6)
    var batchIndex = 0

    batchSupplyList.forEach { supply ->
        var currentDemand = batchDemandList[batchIndex]
        var batchQty = batchQtyList[batchIndex]

        remainBatchQty = (remainBatchQty - batchQty).roundHalfUp(6)
        if (remainBatchQty < 0.0) remainBatchQty = 0.0

        var currentDemandQuantity = currentDemand.demandQuantity

        var fulfillRatio = currentDemandQuantity > Constant.EPS

            : 0.0

                currentDemand,

            fulfill.ppProjectId = context.projectCode()
            fulfill.ppSceneId = context.sceneCode()
            fulfill.frozenFlag = frozenFlag

            toAdd.add(fulfill)

            currentDemand.fulfillQuantity += batchQty
            MaterialSupply.deductGuojiBanchengpingAvailableQty(supply, batchQty)

            if (firstDemand.isFirstConsumeSet) {
                firstDemand.leftRatio -= fulfillRatio
                if (firstDemand.leftRatio < Constant.EPS) firstDemand.leftRatio = 0.0
            }
        }

        batchIndex += 1
    }

    /*

    if (!tempFlag && batchQtyList.isNotEmpty()) {
        var maxSystemNum = workOrder.pieces.maxByOrNull {

        var pieceIndex = 0
        batchQtyList.forEach { batchQty ->
            var pieceQty = batchQty.roundHalfUp(6)
            var selectedMaterialUniqueId = Utils.anyToString(
                batchSupplyList[pieceIndex].uniqueId

                if (pieceIndex == 0) {
                    sourcePiece.quantity = pieceQty
                    sourcePiece.selectedSemiProductUniqueId = selectedMaterialUniqueId
                } else {
                    maxSystemNum += 1
                    var newPiece = workOrder.createNewPiece(pieceQty, maxSystemNum)
                    newPiece.selectedSemiProductUniqueId = selectedMaterialUniqueId
                    toAddPiece.add(newPiece)
                }
            }
            pieceIndex += 1
        }

        if (toAddPiece.isNotEmpty()) {
            context.Group<Piece>().addAll(toAddPiece)
        }

        PlanManager.getInstance(context).splitPiecesToPieceSteps()
    }

    return (totalFulfillRatio, toAdd)
}

return (totalFulfillRatio, toAdd)

## 4. 已注意但未修改的代码点

1. 工序BOM数据按 `queryWorkOrderId` 分组，但使用 `queryOperationBomId` 取值，需要确认两者是否一致。
2. 普通BOM按 `productId+bomVer` 分组，但实际查询BOM只传入 `productId`。
3. 多处分组键直接拼接字段，建议确认不会发生键碰撞；部分替代组键有分隔符，部分没有。
4. `synMaterialSupplyAvailableQuantity` 创建了 `toUpdate`，最后仍执行 `dh.updateBatch(supplies)`。
5. `synInputStepToMaterialDemand` 的删除语句未看到场景过滤条件，需要确认不同场景是否物理隔离。
6. `assignSuppliesToFirstDemands` 中保留了 `pieceId == "A001-10"` 的临时调试代码。
7. 冻结重占统一扣减 `supply.leftQuantity`，是否还需要同步 `leftQuantityVritual`，要结合供应类型确认。
8. 代码中的模型历史字段 `leftQuantityVritual` 拼写应保持不变。
9. 公共供应过滤同时使用 `isGuojiChengpingUseBomBanchengping` 和 `isGuojiBanchengpingMaterialDemand`，两者负责的业务范围需要结合源码确认，避免候选范围与最终过滤条件不一致。
10. 顺位替代料调用公共供应过滤时传入 `firstDemand.demandQuantity` 作为 `leastQty`，会要求单条普通供应达到主需求总量；如果允许多个批次合计满足，可能提前过滤掉合法供应。
11. `assignSuppliesToFirstReplaceDemand1` 中 `totalRemainRatios` 以及循环内的 `totalRemainRatio` 当前没有被使用。
12. `assignSuppliesToFirstReplaceDemand1` 汇总不同需求的供应时未去重，需要确认 `assignAvailableSuppliesToDemand1` 是否自行处理重复供应。
13. 配套组尝试后续优先级时，外层循环没有在 `leftRatio <= Constant.EPS` 后停止，需要确认 `assignSuppliesToDemandSet` 是否会阻止继续分配。
14. `productCompleteQuantityByInv = inputFactor / inputFactor` 在 `inputFactor > 0` 时恒等于1，疑似应使用 `inventoryFulfillQuantity / inputFactor`，需要结合字段业务含义确认。
15. `sendFulfillsToDM` 删除全部 `DMFulfill` 时未看到场景条件，需要确认不同场景是否使用物理隔离表。
16. 24040候选与最终手册过滤使用 `isGuojiBanchengpingMaterialDemand`，其实现要求当前生产产品本身是“B类国际订单”；而生产月份和可用量规则使用 `isGuojiChengpingUseBomBanchengping`，其实现要求“国际成品+BOM半成品”。如果24040目标确实是国际成品使用BOM半成品，这两个判断会导致同一需求走不同分支，需要重点确认。
17. 生产月份过滤直接读取 `workOrder.productionMonth`，但 `workOrder` 可能为空；同一产品有多张工单时只取第一张，也可能取错当前需求所属工单。
18. 库存生产日期使用 `InventoryIn.queryByproductId(...).firstOrNull().productDate`，库存记录为空时存在空指针风险，而且只按产品取第一条库存，未按当前供应批次或唯一标识定位。
19. 生产供应使用配置工序 `firstOrNull().prodcessEndTime`，找不到“Peizhi”工序时存在空指针风险。
20. 月份过滤依赖供应进入方法前 `isFilterByMonth=true`；如果该字段默认是false且没有统一初始化，最后的 `removeIf` 不会过滤任何供应。
21. `assignSuppliesToDemandSet` 在不可混用时直接用 `tempRatio < 1.0` 判断，建议确认是否需要使用 `Constant.EPS` 容差，避免0.999999之类的计算误差。
22. 附件中的整组替代分配代码仍保留“请替换成项目中实际的采购订单/采购申请供应判断方法”注释，需要确认 `isPurchaseOrderSupply` 和 `isPurchaseRequestSupply` 已真实存在，否则代码无法直接编译。
23. `queryProcessingTradeHandBooks` 同时按 `semiFinishedId` 和当前 `materialId` 过滤；如果一本半成品手册允许对应多个候选物料，这里可能只返回当前需求物料，导致调用方无法把其他手册物料加入候选供应。
24. `filterProcessingTradeHandBookSuppliesInPlace` 在找不到任何可用手册库存时会删除全部库存供应，但保留采购和生产计划供应；需确认这符合24040业务规则。
25. `deductGuojiBanchengpingAvailableQty` 调用 `roundHalfDown(7)` 后没有把返回值重新赋给字段；如果Double舍入方法不是原地修改，实际值不会完成七位舍入。
26. `isInventorySupply` 只识别可用库存和待检库存，需要确认半成品库存转换后的 `dataSource` 也属于这两个常量之一。
27. `isPurchaseRequestSupply` 使用常量 `ModelConstants.FromPurchaseRequestc`，末尾的字母c可能是模型中的真实历史命名，也可能是拼写错误，必须按平台常量定义确认。
28. `calcGuojiBanchengpingSortKey` 中仍保留“按实际字段替换”的注释，需要确认 `supply.nr` 和 `demand.batchInventory` 就是最终字段。
29. 三个需求查询方法的分组键均直接拼接字段且没有分隔符，例如 `pieceId+replaceGroupId+consumeSet`，存在不同字段组合生成相同键的风险。
30. `MaterialFulfill.uniqueId` 直接拼接需求ID、供应ID和纳秒时间，没有分隔符；唯一性通常可满足，但不利于排查和人工解析。
