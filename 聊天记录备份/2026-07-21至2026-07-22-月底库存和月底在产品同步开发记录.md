if (!ktMonthlyInventories.isEmpty()) {
    ktMonthlyInventories.forEach{ inventory ->
        if (inventory.expireDate != null && inventory.expireDate.isFinite()) {
            inventory.invalidDate = inventory.expireDate

        log.info(

            "，productionDate：" + inventory.productionDate +
            "，invalidDate：" + inventory.invalidDate +
            "，inboundDate：" + inventory.inboundDate +

- 是否存在`Date.MAX`默认值；
- 是否存在保存前计算公式；
- 是否是只读或计算属性；
- 是否在模型保存阶段覆盖代码赋值。

### 助手

定位到实际执行顺序：

    Excel有6条

    → monthlyInventories为空

            obj.quantity = obj.notLimitQuantity
            monthlyInventories.add(obj)
        }
    }
